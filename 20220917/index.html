学习、教学与教育心理学

学习与教学

今日的学生：多元化、高科技

教师效能感：一个教师坚信他有能力帮助任何一个学生甚至是有学习困难的学生进行学习。

效能感高的老师很少职业倦怠，更易获得满足感。

2010美国《改革蓝皮书：中小学教育法案的重新授权》：

  1. 为学生的升学或就业做好准备
  2. 为每个学校配备优秀的教室
  3. 注重教育公平，保证每个学生机会均等
  4. 逐渐提高标准，鼓励优秀者脱颖而出
  5. 促进教育创新，推进持续性的提高

优质教学：对学生尽职尽责、因材施教、关注学生情感需求、帮助提高自尊心、树立责任感

教育心理学的研究与理论

描述性研究：结果通常包括调查报告、访谈记录、教室里的真实对话片段、课堂活动的录音和录像资料。

相关研究：相关是表明两个事件或者测试结果之间关系强度和方向的数值。

实验研究，准实验研究（被试不是随机分配的，以现存的组为单位参加实验）

单一被试的实验设计：ABAB基线-干预-基线-干预，评估感兴趣的行为并干预-记录研究结果-移除干预处理回到基线-重新进行实验处理

微观发生研究：强调对被试进行详细的观察，探索被试从开始变化到趋于稳定的转变过程，解释变化的深层机制（整个过程、形式多样、不断检验）

时间作用：研究者花费数年观察被试，纵向研究；受限故而更多采用横断研究

《不让一个孩子掉队》基于科学的研究：

  1. 通过系统观察和实验获取可靠而有效的数据
  2. 运用严谨、适当的程序分析数据
  3. 描述详细、具有可重复性
  4. 研究已通过该领域专家严格的独立评审

教育心理学主要目标：理解当一个人在某一情景中教授另一个人某一事情时究竟发生了什么。

实证：基于数据

请区别原理和理论：原理表明两个或多个因素之间的联系—比如教学策略和学生学业成就之间的关系。理论是一组关联的概念，用于解释一组数据并进行预测。原理为回答具体问题提供了许多可能的解释，而理论提供了分析任何情境的观点。研究是一个不断持续的循环过程，包括在一个好理论的基础上提出清晰的研究问题和假设，系统地收集和分析数据，基于数据结果对原有理论进行修正和完善，再基于修正后的理论提出新的更好的问题。

认知发展

认知与脑发展的一般规律

发展：生理、心理、社会性、认知

发展理论关注的三个问题/争论：

  1. 发展的源泉：天性与习性（先天与后天存在复杂的协同作用）
  2. 发展的轨迹：连续性与非连续性
  3. 发展的时期：关键期与敏感期，目前绝大多数关注敏感期（指在一段时间内个体已经准备好经历特定事件或对特定事件做出反应）

发展的一般规律：

  1. 发展速度因人而异
  2. 发展是相对有序的（逻辑顺序）
  3. 发展是逐步发生的

大脑与认知发展有关的特征：偏侧化（两半球的功能特异化）

幼儿的大脑比成人更有可塑性。

皮亚杰主义、信息加工与神经科学

影响发展的因素：成熟（遗传决定）、活动（我们与环境交互、亲身体会）、社会传递

思维的基本倾向：组织与适应（同化和顺应）

图式是思维的基石，这些组织化的动作和思想系统使我们可以在头脑中表征或思考世界中的物体和事件

同化：纳入知识体系，已有图式有效；顺应：调整思维，修正或增加新的图式

认知发展的四个阶段：

  1. 感知运动阶段：客体永久性（不管能否感知，物体始终存在），有逻辑的目的性行为
  2. 前运算阶段：形成和使用符号，很难理解包含逆向思维的问题（如守恒），很难做到去中心化，很难将思维从物体外在表现的直觉中摆脱出来，自我中心（倾向于从自己的想法出发去看待别人眼中的世界）
  3. 具体运算阶段：掌握了守恒、分类、排序等运算能力
  4. 形式运算阶段：能进行假设-演绎推理，青少年自我中心（如”假想的观众感“）

局限性：

  1. 发展阶段被认为是彼此独立的
  2. 低估了儿童的能力
  3. 忽视了文化和社会环境对儿童的重要影响
维果斯基的社会文化观

发展：外部社会活动向内部过程的转化

中心思想：我们特有的心理结构和思维过程来自我们与他人的社会互动，这些社会互动不只影响到认知发展，事实上正是它们创造了我们的认知结构和思维过程。

三个主题：

  1. 个体思维的社会源泉：必须将人类活动置于其文化背景中去理解
  2. 文化工具与认知发展：文化工具尤其是语言是个体发展的关键因素
  3. 最近发展区（独立解决问题时表现出的发展水平和成人或同伴帮助下能达到的发展水平之间的差距）：使学习和发展成为可能的一个区域

皮亚杰：认知发展先于学习，维果斯基：学习是主动过程，不需要被动等待认知

学前阶段 - 小学阶段 - 青春期

发展的社会背景

尤里·布朗芬布伦纳：从社会背景的角度看待个体的发展

生态系统模型：认为影响发展的因素既包括个体内在的生物特点，也包括嵌套结构的社会与文化环境。每个人都生活在一个微观系统中(直接的关系和活动)，这个微观系统包含于中间系统(微观系统中各元素之间的关系)，中间系统又被嵌入
外部系统中(更大的社会环境，如社区)，这三个系统都是宏观系统(文化)的一部分。此外，所有的发展都是发生在特定的时间段内，并受到时间段的影响。因此，生态系统还包括一个历时系统。

背景：环绕在个体的思维、情感和动作的周围，并与个体的这些活动相互作用，最终决定个体的发展和学习的所有环境因素。

自我

埃里克森心理社会发展的八个阶段：

同一性：是指由个体的动机、能力、信念和经历组成的一个一致的自我形象，它涉及深思熟虑地选择和决定，尤其是关于工作、价值观、意识形态、对他人的承诺及看法等方面。

自我概念：指个体对自身的观念、感受、态度和期望等的认识和信念。它是我们尝试向自己解释自己，以此来建构一个关于自己的印象、感觉和信念的图式。

自尊：一种情感反应一一种对自我价值的总体判断，包括个体对自己感到自信和骄傲。
