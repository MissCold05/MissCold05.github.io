《中国教育改造》读书笔记

2022-08-24- 2022-08-26
教研书丛-笔记

《中国教育改造》，陶行知著，这里按照文章记录作者的一些主要思想，其中部分作者的参观访学感悟省略不记。

试验主义与新教育

凡天下之物,莫不有赖于其所处之境况。境况不同，则征象有异。故欲致知穷理,必先约束其境况,而号召其象征，然后效用乃见。此试验之精神,近世一切发明所由来也。彼善试验者立假设,择方法;举凡欲格之物,尽纳之于轨范之中:远者近之,微者大之，繁者简之,杂者纯之,合者析之，分者通之，多方以试之，屡试以验之，更较其异同,审其消长，观其动静,察其变化,然后因果可明而理可穷也。

旧教育：

  1. 依赖天工：役于物而制于天，无自新之力；
  2. 沿袭陈法：徒执古人之成归以解决今之问题；
  3. 率任己意：从事教育之人空想、武断；
  4. 仪型他国：或出于误会，吾所谓新、他以为旧；
  5. 偶尔尝试：计划不确、方法无定

尽刷其旧习，一致以试验为主。

教学合一

重教太过、教学分离

为什么教学合一：

  1. 教师的责任不在教，而在教怎么学
  2. 教的法子根据学的法子
  3. 教师应该一面教一面学，不能固步自封

第一流的教育家

  1. 敢探未发明的新理（创造）
  2. 敢入未开化的边疆（开辟）

学生自治问题之研究

学生结起团体来，大家学习自己管理自己的手续（注意是练习自治，不是放任）

为什么自治：

  1. 国体：共和国需要公民共同自治
  2. 平民主义：需要自由则必须要有自治能力

优点：

  1. 作为修身伦理的实验：养成对公共事情上的愿力、智力、才力
  2. 适应学生需要
  3. 辅助风纪之进步
  4. 促进学生经验之发展

自治不妥当的弊端：

  1. 学生自治当做争权的器具
  2. 学生自治误认为“治人”（驾驭别人的趋势）
  3. 与学校队里
  4. 闹意气
