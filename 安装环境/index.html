安装Ubuntu

卸载

因为好几次都把系统搞坏了，先记一下彻底卸载，个人情况是Ubuntu装在固态硬盘disk0里面

彻底卸载Ubuntu

Ubuntu20.04镜像

在windows里面下载Rufus工具

以ISO镜像写入，完成启动盘的制作。

开机疯狂按F10（依电脑主机品牌可能不同），进入BIOS

设置U盘为第一启动项，进入安装

为了防止目录出问题，一路选英文，最后安完了可以改语言

设置分区

efi系统分区（EFI System Partition），系统启动文件放置的地方，我后面boot也安装在这里。分配100-200MB就行，逻辑分区（Logical），空间起始（Beginning of this space）

swap交换空间，相当于电脑内存（RAM），通常分1-2倍的电脑物理内存大小，但是我觉得8GB已经足够了（我电脑16GB内存），逻辑分区（Logical），空间起始（Beginning of this space）

/ 根分区，文件存储的地方，剩下想要分配的空间都分到这里就行了，主分区（Primary），空间起始（Beginning of this space），Ext4 journaling file system格式

固定内核防止升级

uname -a
